## JSX: 자바스크립트에 마크업 넣기
### 리액트가 마크업과 렌더링 로직을 혼합한 이유
- HTML는 정보, CSS는 디자인, Javascript는 로직을 담당하였으나, 웹이 복잡해짐에 따라 자바스크립트가 HTML을 담당하게 됨. 따라서 리액트에선 컴포넌트에 로직과 마크업을 같이 렌더링하게 된다
